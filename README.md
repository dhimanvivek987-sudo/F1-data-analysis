# Formula 1 Analysis (1950–2020)

## Overview
This project analyzes historical Formula 1 data spanning from 1950 to 2020. The analysis focuses on driver and team performance, points distribution, clustering drivers into performance tiers, and forecasting team points using Prophet.

The project is implemented in Python using pandas, NumPy, Matplotlib, Seaborn, scikit-learn, and Prophet.

---

## Dataset
The dataset is sourced from Kaggle: [Formula 1 World Championship 1950–2020](https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020)

Key files used:
- `drivers.csv`, `constructors.csv`, `races.csv`, `results.csv`, `circuits.csv`
- Additional files: `constructor_results.csv`, `driver_standings.csv`, etc.
